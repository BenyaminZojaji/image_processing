## Sudoku Detector
- Detect sudoku and transform to flat image using *imutils.perspective*
```shell
usage: sudoku_detector.py [-h] [--input INPUT] [--output OUTPUT] [--kernel_size KERNEL_SIZE] [--color COLOR] [--draw DRAW]
```
